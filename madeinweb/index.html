What we can learn

HTML

Hyper Text Markup Language = HTML
ホームページを作る上で一番基本となる内容の構造を作成するための言語となります。
ページの基本構造となるとともに使用するテキストや画像などもHTMLで配置してゆきます。
Web制作する上では不可欠な言語です。

CSS

Cascading Style Sheet = CSS
HTMLでマークアップしたデータを見やすくレイアウトするためにはスタイルシートを使用します。
CSSを理解すれば、いろいろなレイアウトを表現できるようになります。
